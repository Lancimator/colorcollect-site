Group mode

One color. Six letters. Everyone hunts.

Spin up a room, share a 6-letter code, and turn any color into a live scavenger hunt with friends. Everyone shoots the same color, the best photos win, and you all get one shared montage. Free, no accounts to join.

How it works

  1. Create a room and pick the colorThe host chooses the target color and grid size, and can name the room. ColorCollect hands back a 6-letter join code.
  2. Share the code or invite linkFriends join with the 6-letter code or a tap-to-open invite link — no account signup, just a display name.
  3. Everyone hunts the real world, liveYou all chase the same color at once, and every shot lands on the shared grid in real time on everyone's phone.
  4. Vote for the best shot per cellIn Vote mode anyone can submit photos for any cell; when everyone's finished, a round opens and you all pick the winner for each cell.
  5. One montage, plus final standingsThe finished montage is built from everyone's winning shots — and a shareable podium card tallies who landed the most cells.

Two ways to play

Pick the vibe when you create the room — split the work, or let the group vote.

Assigned cells

Split the grid; each person shoots their own cells.

  • The grid splits evenly between everyone — the host mops up any leftover cells.
  • Claim the empty cells you want; you only ever shoot the cells you own.
  • No two people share a cell, so there's zero overlap.
  • The host builds the montage once everyone's marked ready.

Vote

Everyone submits shots, then the group votes on the best.

  • Anyone can submit a photo for any cell, so a cell can gather several takes.
  • When everyone's finished, a voting round opens for the whole group.
  • The most-voted photo wins each cell; ties go to the earliest upload.
  • The host finalizes voting to build the montage from the winners.
